Concept Extraction Pipeline (CS 1980 Capstone)

A full NLP pipeline designed to extract instructional concepts from university lecture slides using rule-based annotation, LLM prompting, and regression-driven analysis of model performance. Includes multimodal prompting, codebooks, filtering stages, structural metrics, and F1 evaluation.

LLM Compression & Bias Study (Replication Project)

A replication and statistical extension of Gonçalves & Strubell (2023), examining how quantization and distillation affect bias in BERT-family models. The project applies OLS, significance testing, and effect-size analysis.

PPG Paints Supervised Learning Project

A machine learning project evaluating predictive performance of multiple models using PPG color data. Includes logistic regression, neural networks, regularization, ROC analysis, and cross-validation. Completed for CS 1675.

Bias-Aware Toxicity Classifier (CS 1684 Project)

A full offensive-language classification and fairness auditing pipeline. This project compares PerspectiveAPI toxicity scores to a TF-IDF + logistic regression model, evaluates macro-F1 performance, and analyzes dialect-specific false-positive rates using TwitterAAE demographic tags.
